# 交易所配置

## 支持的交易所
- Binance
- OKX
- Bybit
- Gate.io

## 配置方式

### 方式1: API Key
在对应交易所创建 API Key，确保开启：
- 合约交易权限
- 读取权限

### 方式2: 环境变量
```bash
# Binance
export BINANCE_API_KEY="your_api_key"
export BINANCE_SECRET_KEY="your_secret_key"

# OKX
export OKX_API_KEY="your_api_key"
export OKX_SECRET_KEY="your_secret_key"
```

## ⚠️ 安全建议
1. 不要提交 API Key 到 Git
2. 使用只读权限的 Key 优先
3. 设置 IP 白名单
4. 定期更换 Key
